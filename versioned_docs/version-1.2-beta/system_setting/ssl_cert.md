# SSL证书

**APIPark** 支持上传 **SSL证书** 实现 **HTTPS协议通信**。

## 添加证书

进入 **系统设置** 模块，在侧边栏选择 **SSL 证书**，点击 **添加证书**，在弹窗中填写：

- 密钥（`.key`）
- 证书（`.pem`或`.csr`）

点击 **确认** 即可。
